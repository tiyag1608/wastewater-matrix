# Wastewater Matrix

## Overview
Wastewater Matrix is a React-based SCADA simulation dashboard that demonstrates the implementation of Data Structures and Algorithms in a real-world municipal wastewater monitoring system.

## Features
- Chemical Search (Trie)
- Valve Control (Stack)
- Sample Queue (Queue)
- Sensor Registry (HashMap)
- Overflow Sorter (Heap)
- Pipeline Network (Graph)
- Gravity Planner (BFS & Dijkstra)
- Chemical Balancer (Greedy Algorithm)

## Tech Stack
- React 18
- Vite
- JavaScript
- CSS
- GSAP

## Installation
